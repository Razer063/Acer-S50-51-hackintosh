## 安装部分

- #### 安装教程

  - `F2`进入BIOS,
  - 先查看

- 安装之后操作

  - 安装好系统之后，先用制作好的EFI文件进入系统

  - `终端`执行以下代码

  - ```
    sudo spctl --master-disable
    ```

  - 再执行重建缓存：

  - ```
    sudo kextcache -i /
    ```

  - 完整替换EFI

  - 最后重启完成
